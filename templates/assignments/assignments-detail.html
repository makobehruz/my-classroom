Курс: Python Backend

Описание задания

В этом задании вы будете практиковать концепцию полиморфизма в Python. Вам нужно будет создать несколько классов с общим интерфейсом и продемонстрировать, как они могут быть использованы полиморфно.

Требования

  • Создайте базовый класс Shape с методом area()
  • Создайте подклассы Circle, Rectangle и Triangle
  • Реализуйте метод area() для каждого подкласса
  • Продемонстрируйте использование полиморфизма с этими классами
  • Напишите функцию, которая принимает список фигур и вычисляет общую площадь
